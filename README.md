# Augusto Santana | Portfolio Web Premium

Portfolio em React para apresentar demos navegaveis, templates comerciais e a demo TCG do acervo atual.

## Visao geral

- Hero editorial e minimalista
- Showcase principal com templates para nichos diferentes
- Demos ativas organizadas na mesma experiencia do portfolio
- Estrutura simples, componentizada e pronta para evoluir

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Estrutura

```text
src/
  components/
    ContactBlock.jsx
    Footer.jsx
    Hero.jsx
    ProjectCard.jsx
    ProjectTemplateVisual.jsx
    ProjectsShowcase.jsx
    SectionHeader.jsx
    SignatureStrip.jsx
    ThemeToggle.jsx
  data/
    demos.js
    projects.js
    site.js
  App.jsx
  main.jsx
  styles.css
```

## Direcao do projeto

O site foi desenhado para comunicar maturidade, clareza e capacidade de entrega sem carregar sobras de projetos antigos. O foco esta na apresentacao de demos e templates como produtos reais, com composicoes grandes, copy curta e espaco para evoluir novos trabalhos.
